TO BUILD NGINX WEBSERVER ---> The html was took in a random Github


First step --> Build the Dockerfile:
	sudo docker build -t [name_futur_image] .

Build in the repository where Dockerfile is

Second step --> Run the image to create the container
	sudo docker run --name [name_actual_image] -d -p 8080:80 [name_container]

Now check if all is good (sudo docker images, sudo docker ps) if error check the logs (sudo docker logs [first 3 characters of the id container])
