DANS JENKINS POUR PROMETHEUS -----------------------
J'ai créer une pipeline avec hello-world juste pour avoir des metrics 

ensuite j'ai installé le plugins Prometheus metrics
Puis dans 'Manage Jenkin' > 'SYSTEM' j'ai fait la config
dans path il y a 'prometheus' c'est le endpoint ou prometheus va scrape les données
pour y accéder c'est : 'localhost:8081/prometheus' 

Ensuite je modifie le 'prometheus.yml' pour ajouter /prometheus
