NE SUPPRIMER AUCUN DES DEUX DOSSIERS

Docker Jenkins pour l'orchestreur ( docker ccompose ect.. )
Docker Webserv pour la page web ( Nginx )
Prometheus pour la recolte de métrique de Jenkins ( donc données système )
Gravana pour récupérer les données de Prometheus, ensuite créer des visuels pour une meilleurs compréhension ( diagramme, tableau, ect...)

sudo docker compose up -d
sudo docker compose ls

créer le fichier /var/lib/grafana pour avoir l'acces
